UI Prototype

This template enables you to quickly generate a single page HTML prototype application that is actually a functioning application and the basis of producing an end product.

UI Prototype uses HTML 5, Bootstrap 3 for CSS styles, JQuery 2 for Javascript.

UI Prototype - Intro

The basis of this HTML template are individual pages that are represented in <section> tags. Each section has an id and is defined as hidden.

<section id="intro" style="display:none">
</section> <!-- intro -->

To render any given page defined as a section you add to a standard bootstrap link or button the "render" class and you define a data-section element defining the section you wish to render. On the home page you clicked a button to get to this intro section. 

<button class="btn btn-success render" data-section="intro">Intro<button>

Behind the scenes there is ajQuery click() function for the defined .render class. This takes care of the appropriate showing of the section. It does a lot more also, we discuss that later.

UI Prototype - Layout

Bootstrap enables a responsive CSS design of your webpage. You can customize this for your application as you wish. The following is the default that is used here and works to provide a view for three responsive layouts (cell phone, tablet, computer). 

<section id="layout" style="display:none">
  <div class="container-fluid">
    <div class="row-fluid row-centered">
      <div class="col-xs-12 col-sm-8 col-md-6 col-sm-offset-2 col-md-offset-3">
        <h1>UI Prototype - Layout</h1>
        <p>...</p>
      <div> <!-- col -->
    <div> <!-- row -->
  <div> <!-- container -->
</section> <!-- layout -->

You can resize this window down to see the responsive design. Code blocks are not the usual application so this becomes less effective in a smaller cell phone size resolution.
